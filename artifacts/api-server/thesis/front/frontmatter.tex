\vspace{0.8cm}
{\normalsize
Миний бие Эрдэнэтөгсийн Очбадрах, ``РЕСТОРАНЫ ШИРЭЭ ЗАХИАЛГА, ЦЭС УДИРДЛАГЫН МОБАЙЛ
СИСТЕМИЙН ЗАГВАРЧЛАЛ, ХЭРЭГЖИЛТ'' сэдэвт энэ ажил нь минийх бөгөөд дараахыг нотолж байна.
Үүнд:

\begin{itemize}[leftmargin=2cm]
  \item Горилогч энэ ажлыг тус сургуулиас боловсролын зэрэг авахаар бүхэлд нь буюу
        голлон хийсэн болно.
  \item Энэ ажлын аль нэг хэсгийг тус сургуульд эсвэл өөр байгууллага боловсролын
        зэрэг, мэргэшил авахаар өмнө нь илгээсэн бол түүнийгээ тодорхой заасан болно.
  \item Бусад хүмүүсийн хэвлүүлсэн ажлаас зөвлөгөө авсан бол түүнийгээ үндэслэсэн болно.
  \item Бусад хүмүүсийн ажлаас ишлэл хийхдээ гол эх үүсвэрийг нь заасан болно.
  \item Миний ажилд тусалсан голлох бүх эх үүсвэрт талархаж байна.
  \item Ажлыг бусадтай хамтарсан бол алийг нь бусад хүмүүс хийсэн болохыг тодорхой
        заасан болно.
\end{itemize}

{\normalsize Рестораны ширээ захиалга, цэс удирдлагын мобайл системийн загварчлал, хэрэгжилт}\\[6pt]
{\normalsize Э.~Очбадрах}\\
{\small \texttt{ochkoo187@gmail.com}}

\vspace{0.8cm}
{\normalsize
Хүнсний үйлчилгээний салбар нь дэлхий нийтэд хурдацтай дижитал шилжилт хийж буй
өнөө үед Монгол Улсын рестораны 92\%-д шинэ технологийг нэвтрүүлэхэд санхүүгийн
дарамт, техникийн мэдлэгийн дутагдал, Монгол хэлийг дэмждэг бэлэн шийдлийн
байхгүй байдал зэрэг саад тотгор хэвээр байна.

Энэхүү дипломын ажлын хүрээнд рестораны ширээ тус бүрт байрлуулсан QR кодыг
үйлчлүүлэгч ухаалаг утсаараа уншуулж цэстэй танилцан захиалга өгөхөд тухайн
захиалга нь WebSocket протоколоор ажилтны дэлгэцэнд бодит цаг хугацаанд дамждаг
бүрэн автоматжсан мобайл системийг загварчилж хэрэгжүүлсэн. Системд мөн ширээ
захиалга, цэс удирдлага, захиалгын статус хяналт, тайлангийн модулиудыг нэгтгэсэн.

Системийг React 18 + TypeScript (вэб клиент), Node.js/Express.js (API сервер),
PostgreSQL 16 (өгөгдлийн сан), Socket.IO v4 (бодит цагийн холболт) технологиудыг
ашиглан 4 давхаргат архитектураар хөгжүүлсэн. JWT HS256 токен дээр суурилсан хандалтын
хяналт, Drizzle ORM-д суурилсан өгөгдлийн давхарга, pnpm workspace монорепо бүтцийг
нэвтрүүлсэн.

Туршилтын үр дүнд API хариу 120 мс дундаж, WebSocket мессежийн дамжуулалт 45 мс,
SUS (System Usability Scale) үнэлгээ 84.2/100 оноо (``Маш сайн'' зэрэглэл), хэрэглэгч
захиалга өгөхөд зарцуулах хугацаа уламжлалт аргаас 62\%-иар буурсан үр дүн гарсан.
}

\begin{center}
{\large\bfseries Abstract}
\addcontentsline{toc}{chapter}{Abstract}

\vspace{0.3cm}
{\normalsize Modeling and Implementation of a Mobile System for Restaurant Table Booking and Menu Management}\\[6pt]
{\normalsize E.~Ochbadrakh}\\
{\small \texttt{ochkoo187@gmail.com}}
\end{center}

\vspace{0.8cm}
{\normalsize
The food service industry is undergoing rapid digital transformation worldwide. In
Mongolia, 92\% of restaurants face barriers to adopting new technologies, including
financial constraints, lack of technical expertise, and the absence of ready-made
solutions supporting the Mongolian language.

This thesis presents the design and implementation of a fully automated mobile system
for restaurant table booking and menu management. Customers scan a QR code placed on
each table using their smartphones to instantly access the restaurant's digital menu,
place orders, and book tables. Orders are transmitted in real-time to the staff
dashboard via WebSocket protocol. The system also integrates table reservation, menu
management, order status tracking, and reporting modules.

The system was developed using a 4-layer architecture with React 18 + TypeScript
(web client), Node.js/Express.js (API server), PostgreSQL 16 (database), and
Socket.IO v4 (real-time communication). Authentication is handled via JWT HS256
tokens, the data layer uses Drizzle ORM, and the project follows a pnpm workspace
monorepo structure.

Testing results demonstrated an average API response time of 120~ms, WebSocket
message delivery of 45~ms, a System Usability Scale (SUS) score of 84.2/100
(``Excellent'' rating), and a 62\% reduction in order placement time compared to
traditional methods. Load testing with k6 confirmed stable performance under 200
concurrent connections.
}

\vspace{0.5cm}
{\small\textbf{Keywords:} QR code, mobile ordering, restaurant system, table booking,
menu management, WebSocket, Socket.IO, real-time notification, React, Node.js, PostgreSQL}

{\normalsize
Энэхүү дипломын ажлыг амжилттай гүйцэтгэхэд маш олон хүмүүсийн дэмжлэг,
туслалцаа шаардагдсан бөгөөд тэдэнд талархал илэрхийлэхийг хүсч байна.

Юуны өмнө дипломын ажлын санааг олох, чиглэл зааж өгөх, хэрэгжүүлэх үр дүнг
боловсруулах, нэмэлт мэдээлэл өгөн санал солилцох зэргээр хамтран ажиллаж
туслалцаа үзүүлсэн удирдагч багш П.~Нарангэрэл-д гүнээ талархсанаа илэрхийлье.

Мөн зөвлөх багш Доктор Б.~Намжилдорж-д судалгааны арга зүй, шинжилгээний чиглэлээр
өгсөн үнэтэй зөвлөмжүүдэд нь баярлалаа.

Туршилтын явцад оролцож санал хүсэлтээ өгсөн рестораны хэрэглэгчид болон ажилтнуудад
талархал илэрхийлье. Тэдний санал хүсэлт системийг улам боловсронгуй болгоход чухал
үүрэг гүйцэтгэсэн.

Эцэст нь сурч буй хугацаанд дэмжлэг үзүүлсэн Өмнөговь аймаг дахь Технологийн Дээд
Сургуулийн багш нар болон хамт суралцагсаддаа баярлалаа.
}
